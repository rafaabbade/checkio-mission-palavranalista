Enunciado

Carlos, um professor de idiomas, está ajudando seus alunos a praticar vocabulário. Ele precisa de uma função que verifique se uma palavra específica está presente em uma lista de palavras fornecida pelos alunos.

Crie uma função encontra_palavra que recebe uma lista de palavras e uma palavra alvo como parâmetros, e retorna True se a palavra alvo está presente na lista e False caso contrário.

Entrada: Uma lista de palavras e uma palavra alvo.

Saída: True se a palavra alvo está presente na lista e False caso contrário.

Exemplo:

{% if interpreter.slug == "js-node" %} 
countConsecutiveSummers(42) == 4
countConsecutiveSummers(99) == 6
{% else %} 
encontra_palavra(['python', 'java', 'ruby', 'c++'], 'python') >>> True
encontra_palavra(['gato', 'cachorro', 'pássaro', 'leão'], 'elefante') >>> False
encontra_palavra(['azul', 'vermelho', 'verde', 'amarelo'], 'rosa') >>> False
{% endif %}
